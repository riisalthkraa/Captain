Politique de Confidentialité

Comment Cap'taine protège vos données et celles de vos enfants

En résumé

Cap'taine (Version Famille) ne collecte AUCUNE donnée sur vous ou vos enfants.

  • Toutes les données restent sur votre ordinateur
  • Nous n'avons pas de serveur qui reçoit vos informations
  • Pas de tracking, pas d'analytics, pas de publicité
  • Vous avez un contrôle total sur vos données

Cette politique détaille le fonctionnement complet pour être totalement transparent.

1. Qui sommes-nous ?

Cap'taine est développé par :

Responsable du traitement : VIEY David

Email : Riisalthkraal@gmail.com

Adresse : Castillon-la-Bataille (33350), Gironde, France

Pour toute question relative à vos données personnelles, vous pouvez nous contacter à l'adresse ci-dessus.

2. Données collectées par l'Application (Version Famille)

2.1. Ce que nous NE collectons PAS

La Version Famille de Cap'taine ne transmet AUCUNE donnée à nos serveurs car nous n'en avons pas.

  • ❌ Pas de création de compte obligatoire
  • ❌ Pas de télémétrie
  • ❌ Pas d'analytics
  • ❌ Pas de tracking publicitaire
  • ❌ Pas de partage avec des tiers
  • ❌ Pas de profilage

2.2. Ce qui est stocké LOCALEMENT sur votre ordinateur

Les données suivantes sont stockées uniquement sur votre ordinateur, dans le dossier de l'application :

Donnée Finalité Durée de conservation
Prénom de l'enfant Personnaliser l'expérience ("Bonjour Lucas !") Jusqu'à suppression du profil
Niveau scolaire Adapter les exercices au niveau Jusqu'à suppression du profil
Historique des exercices Suivre la progression, adapter la difficulté Jusqu'à suppression du profil
Réponses données Analyser les erreurs, personnaliser l'aide Jusqu'à suppression du profil
Statistiques de session Afficher les tableaux de bord Jusqu'à suppression du profil
Préférences (thème, sons) Mémoriser vos choix Jusqu'à désinstallation
Clés API (si configurées) Connexion aux services IA Jusqu'à suppression manuelle

2.3. Emplacement des données

Les données sont stockées dans :

  • Windows : %APPDATA%\captaine
  • macOS : ~/Library/Application Support/captaine
  • Linux : ~/.config/captaine

Vous pouvez accéder à ces dossiers, les sauvegarder ou les supprimer à tout moment.

3. Cas particulier : utilisation de providers IA tiers

Important

Si vous choisissez d'utiliser un provider IA cloud (OpenAI, Anthropic, Google), les messages de conversation seront envoyés à leurs serveurs pour traitement.

3.1. Ollama (recommandé)

Si vous utilisez Ollama comme provider IA :

  • ✅ Tout reste 100% local sur votre ordinateur
  • ✅ Aucune donnée n'est envoyée sur internet
  • ✅ C'est la configuration que nous recommandons

3.2. OpenAI / Claude / Gemini

Si vous configurez une clé API pour ces services :

  • Les messages de conversation sont envoyés à leurs serveurs
  • Ces données sont soumises aux politiques de confidentialité de ces entreprises
  • Nous vous recommandons de lire leurs politiques respectives

Liens vers leurs politiques :

  • OpenAI Privacy Policy
  • Anthropic Privacy Policy
  • Google Privacy Policy

3.3. Ce que nous ne faisons jamais

  • Nous ne stockons pas vos clés API sur nos serveurs (nous n'en avons pas)
  • Nous ne faisons pas transiter les données par un proxy
  • Nous n'avons pas accès à vos conversations avec l'IA

4. Données collectées par le site web

4.1. Navigation sur le site

Notre site web ne collecte aucune donnée de navigation :

  • Pas de Google Analytics ni autre outil de tracking
  • Pas de cookies publicitaires
  • Pas de pixels de suivi

4.2. Formulaire de contact

Si vous utilisez notre formulaire de contact, nous collectons :

Donnée Finalité Base légale Conservation
Nom Vous identifier pour répondre Consentement 3 ans
Email Vous répondre Consentement 3 ans
Message Traiter votre demande Consentement 3 ans

Ces données sont utilisées uniquement pour répondre à votre demande et ne sont jamais partagées avec des tiers.

4.3. Cookies

Le site utilise uniquement des cookies techniques strictement nécessaires :

  • Cookie de session : maintient votre navigation (expire à la fermeture du navigateur)
  • Cookie de préférence : mémorise vos choix comme le thème (expire après 1 an)

Ces cookies ne nécessitent pas votre consentement car ils sont indispensables au fonctionnement du site.

5. Protection des données des enfants

Notre engagement

Cap'taine est destiné aux enfants. Leur protection est notre priorité absolue.

5.1. Pas de collecte de données d'enfants

La Version Famille ne collecte ni ne transmet aucune donnée personnelle des enfants. Les informations (prénom, niveau, progression) restent uniquement sur l'ordinateur familial.

5.2. Supervision parentale

Nous recommandons aux parents de :

  • Superviser la configuration initiale de l'application
  • Vérifier les paramètres de provider IA
  • Accompagner l'enfant lors des premières utilisations

5.3. Pas de publicité ni de liens externes

L'application ne contient :

  • Aucune publicité
  • Aucun achat intégré
  • Aucun lien vers des sites externes (hors documentation)
  • Aucune fonctionnalité sociale ou de partage

6. Vos droits (RGPD)

Conformément au Règlement Général sur la Protection des Données (RGPD), vous disposez des droits suivants :

Droit Description Comment l'exercer
Accès Obtenir une copie de vos données Données locales : accédez au dossier de l'app
Site : contactez-nous
Rectification Corriger des données inexactes Modifiez directement dans l'application
Effacement Supprimer vos données Supprimez le profil ou désinstallez l'app
Portabilité Récupérer vos données Copiez le dossier de données de l'app
Opposition S'opposer au traitement Contactez-nous (site) ou cessez d'utiliser (app)
Limitation Limiter le traitement Contactez-nous

6.1. Comment exercer vos droits

Pour l'application : vos données sont déjà sous votre contrôle total (stockage local).

Pour le site web : envoyez un email à Riisalthkraal@gmail.com avec :

  • Votre demande précise
  • Une preuve d'identité

Nous répondrons dans un délai de 30 jours.

6.2. Réclamation

Si vous estimez que vos droits ne sont pas respectés, vous pouvez déposer une réclamation auprès de la CNIL :

www.cnil.fr

7. Sécurité des données

7.1. Application

Les données locales de l'application sont aussi sécurisées que votre ordinateur. Nous recommandons :

  • De protéger votre ordinateur par un mot de passe
  • De maintenir votre système à jour
  • D'utiliser un antivirus

7.2. Site web

Notre site web utilise le protocole HTTPS pour chiffrer les communications.

7.3. Clés API

Si vous configurez des clés API pour des services tiers, elles sont stockées localement et ne sont jamais transmises à nos serveurs.

8. Transferts internationaux

Version Famille : Aucun transfert. Tout reste sur votre ordinateur.

Si vous utilisez un provider IA cloud :

  • OpenAI : données transférées aux États-Unis
  • Anthropic : données transférées aux États-Unis
  • Google : données transférées selon leur politique

Ces entreprises disposent de mécanismes de transfert conformes au RGPD (clauses contractuelles types, etc.).

9. Modifications de cette politique

Nous pouvons mettre à jour cette politique de confidentialité. En cas de modification substantielle :

  • Une notification sera affichée dans l'application
  • La date de mise à jour sera indiquée sur cette page

Nous vous encourageons à consulter régulièrement cette page.

10. Contact

Pour toute question relative à cette politique de confidentialité :

  • Email : Riisalthkraal@gmail.com
