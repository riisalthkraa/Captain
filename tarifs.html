Questions fréquentes sur les tarifs

Pourquoi la version Famille est-elle gratuite ?

Nous croyons que l'éducation doit être accessible à tous. La version Famille est financée par la future version Enseignement payante. C'est un modèle classique : gratuit pour les particuliers, payant pour les professionnels.

Y aura-t-il de la publicité ?

Non, jamais. Nous ne mettrons jamais de publicité dans une application destinée aux enfants. Notre modèle économique repose uniquement sur la version Enseignement.

La version gratuite sera-t-elle limitée à l'avenir ?

Non. Toutes les fonctionnalités actuellement gratuites le resteront. Nous pourrions ajouter des fonctionnalités "premium" mais jamais retirer ce qui existe déjà.

Je suis professeur, puis-je utiliser la version gratuite ?

Oui ! La version Famille est parfaite pour un usage personnel ou pour recommander aux parents de vos élèves. La version Enseignement sera utile si vous voulez gérer une classe entière et suivre la progression de chaque élève.

Pourquoi la version Enseignement nécessite une connexion ?

La synchronisation entre le tableau de bord professeur et les comptes élèves nécessite un serveur central. Les données seront chiffrées et hébergées en France/Europe (RGPD compliant).

Y aura-t-il des tarifs Éducation Nationale ?

Oui, nous prévoyons des tarifs spéciaux pour les établissements publics et les contrats collectifs. Contactez-nous pour en discuter.
