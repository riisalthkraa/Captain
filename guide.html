🚀 Guide de démarrage

Premiers pas avec Cap'taine - Moins de 2 minutes pour commencer l'aventure

1️⃣

Mode Invité : Démarrage rapide

Au premier lancement :

  1. Cap'taine te propose "Mode Invité" ou "Créer un compte"
  2. Clique sur "Mode Invité" pour tester immédiatement
  3. Tu pourras créer ton profil plus tard !

💡 Le mode invité permet d'utiliser Cap'taine sans créer de profil. Tes progrès ne seront pas sauvegardés.

2️⃣

Créer ton profil (recommandé)

Pour sauvegarder ta progression :

  1. Clique sur l'icône ⚙️ Paramètres (en haut à droite)
  2. Entre ton prénom (ex: "Léa", "Tom")
  3. Choisis ta classe actuelle (CP, CE1, CE2, CM1, CM2, 6ème, 5ème, 4ème, 3ème)
  4. Clique sur "Enregistrer"

✅ Tes badges, ton XP, ton arbre de progression et tes statistiques seront maintenant sauvegardés !

💡

ASTUCE : Accède à TOUS les niveaux

Tu n'es PAS limité à ta classe !

Dans ⚙️ Paramètres, tu peux changer de classe à tout moment pour :

  • 📚 Réviser des notions plus simples : Tu es en CM2 mais veux revoir les additions du CP ? Change ta classe en "CP" !
  • 🚀 Essayer des exercices plus difficiles : Tu es en CE1 mais tu veux défier la 6ème ? Change ta classe en "6ème" !
  • 🎯 Travailler une matière spécifique : Tu veux faire tous les exercices de maths du CM1 ? Change en "CM1" !

🎓 Cap'taine = 2000+ exercices de TOUS les niveaux (CP à 3ème)
Change de classe autant de fois que tu veux. Ta progression est sauvegardée pour chaque niveau !

3️⃣

Choisir une activité

4 modes de jeu :

  • 💬 Chatbot IA : Discute avec Cap'taine, pose des questions, génère des exercices personnalisés (nécessite une clé API ou version desktop avec Ollama)
  • 📚 Exercices : 2000+ exercices dans toutes les matières (maths, français, sciences, histoire, langues...)
  • 🎮 Mini-jeux : 13+ jeux éducatifs (Course de Calcul, Pizza des Fractions, Memory, etc.)
  • ⚔️ VS Cap'taine : Défi contre l'IA sur un thème de ton choix
🌐

Version Web vs Version Desktop

Version Web (dans le navigateur) :

  • ✅ Tous les 2000+ exercices
  • ✅ Tous les 13+ mini-jeux
  • ✅ Dashboard, badges, XP, arbre de progression
  • ✅ Chatbot IA avec clés API (OpenAI, Claude, Gemini, Mistral, Groq)
  • ❌ Ollama NE FONCTIONNE PAS en version web (même si installé sur ton PC)
  • ⚠️ Sauvegarde dans le navigateur (si tu vides le cache, tu perds ta progression)

Version Desktop (à télécharger - .exe) :

  • ✅ Toutes les fonctionnalités de la version web
  • ✅ Support Ollama (IA 100% locale et gratuite)
  • ✅ Sauvegarde permanente sur ton PC (même si tu réinstalles)
  • ✅ Meilleures performances
  • ✅ Fonctionne 100% hors ligne (sans internet)

⚠️ IMPORTANT pour Ollama :
Même si Ollama est installé sur ton PC, la version web dans le navigateur ne peut PAS l'utiliser pour des raisons de sécurité.
Pour utiliser Ollama, tu DOIS télécharger et installer la version desktop de Cap'taine.

4️⃣

Gagner de l'XP et débloquer des badges

Comment progresser :

  • 🎯 Chaque exercice réussi = 15-30 XP
  • 🎮 Chaque mini-jeu terminé = 15-30 XP
  • 📈 100 XP = 1 niveau (pas de limite)
  • 🏆 55 badges à débloquer (premiers pas, séries, maîtrises, spéciaux...)
  • 🌳 Ton arbre évolue : 🌱 → 🌿 → 🌳 → 🌲 → 🌴

📊 Consulte ton Dashboard pour voir tes stats, tes forces et tes faiblesses !
