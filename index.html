Comment Cap'taine fonctionne vraiment

Pas de bullshit marketing. Voici exactement ce qui se passe sous le capot.

📊

Système adaptatif

Ce que c'est : Des algorithmes éprouvés (pas du "machine learning" magique)

  • Algorithme SM-2 pour la répétition espacée (le même qu'Anki)
  • 40+ règles de détection de patterns d'erreurs
  • Analyse du temps de réponse et du taux de réussite

Ces méthodes sont scientifiquement validées et parfaitement adaptées à l'apprentissage scolaire.

🤖

Intelligence Artificielle

4 options :

  • Ollama — IA locale, gratuite, rien ne sort de votre PC ✅
  • OpenAI/Claude/Gemini — Si vous avez une clé API
  • Sans IA — La banque d'exercices intégrée suffit

L'IA sert à générer des exercices et des explications personnalisées. Le système adaptatif fonctionne même sans.

💾

Vos données

Où elles vont : Nulle part. Elles restent chez vous.

  • Stockage local dans le dossier de l'app
  • Nous n'avons aucun serveur
  • Pas de télémétrie, pas d'analytics
  • Mode invité = aucune trace

Exception : si vous utilisez une IA cloud (OpenAI etc.), les messages vont à leur API. Utilisez Ollama pour rester 100% local.

Motivation & Suivi

Un dashboard qui track tout (vraiment)

Chaque exercice, chaque badge, chaque jour. Sauvegardé localement sur ton PC.

🏆

55 Badges à débloquer

Exemples de badges :

  • 🌟 Premier pas : Termine ton premier exercice
  • 🏆 Champion : Atteins le niveau 10
  • 🔥 Série de 7 : 7 bonnes réponses d'affilée
  • 🧮 As des maths : 100% à 5 exercices de maths
  • 🎮 Boss final battu : Bats Cap'taine en mode VS
  • 🌙 Couche-tard : Travaille après 22h

Chaque badge débloqué reste acquis. L'enfant peut choisir 3 badges favoris à afficher sur son profil.

🌳

Arbre de progression

Évolution visuelle :

  • 🌱 Niveau 1-4 : Graine qui pousse
  • 🌿 Niveau 5-9 : Jeune plante
  • 🌳 Niveau 10-14 : Arbre
  • 🌲 Niveau 15-19 : Grand arbre
  • 🌴 Niveau 20+ : Arbre majestueux

Chaque exercice donne de l'XP. 100 XP = 1 niveau. Pas de limite de niveau.

🎮

13+ Mini-jeux

Pour tous les niveaux :

  • 🏎️ Course de Calcul : Résous vite
  • 🍕 Pizza des Fractions : Apprends les fractions
  • ⚔️ Bataille des Tables : Duel contre Cap'taine
  • 🔍 Explorateur de Mots : Mots cachés
  • 🃏 Memory des Calculs : Associe calculs/résultats
  • 🕐 L'Horloge Magique : Apprends l'heure

Chaque jeu track le meilleur score, meilleur temps et historique complet.

🎯

Analyse ML des compétences

Le système détecte automatiquement :

  • ✅ Forces : Compétences > 75% de réussite
  • ⚠️ Faiblesses : Compétences < 50% de réussite
  • 📊 En cours : Compétences entre 50-75%
  • 🔍 Patterns d'erreurs : Top 5 types récurrents

Recommandations priorisées : Cap'taine propose les notions à travailler en priorité.

🔥
