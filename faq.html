Foire Aux Questions

Tout ce que vous devez savoir sur Cap'taine

Général

Cap'taine est un assistant éducatif intelligent sous forme d'application desktop (Windows, macOS, Linux). Il accompagne les enfants du CP à la 3ème dans leur apprentissage.

Deux modes d'utilisation :

  • Banque d'exercices : 2000+ exercices pré-enregistrés (toutes matières, tous niveaux)
  • Chatbot IA : Génère des exercices personnalisés en temps réel

L'app fonctionne 100% localement. Aucune donnée n'est envoyée sur internet (sauf si vous utilisez une IA cloud comme OpenAI).

Version Web (dans le navigateur) :

  • ✅ Tous les 2000+ exercices
  • ✅ Tous les 13+ mini-jeux
  • ✅ Dashboard, badges, XP, arbre de progression
  • ✅ Chatbot IA avec clés API (OpenAI, Claude, Gemini, Mistral, Groq)
  • ❌ Ollama NE FONCTIONNE PAS (même si installé sur ton PC)
  • ⚠️ Sauvegarde dans le navigateur (si tu vides le cache, tu perds ta progression)

Version Desktop (à télécharger - .exe/.dmg/.AppImage) :

  • ✅ Toutes les fonctionnalités de la version web
  • ✅ Support Ollama (IA 100% locale et gratuite)
  • ✅ Sauvegarde permanente sur ton PC
  • ✅ Meilleures performances
  • ✅ Fonctionne 100% hors ligne (sans internet)

⚠️ IMPORTANT pour Ollama :
Même si Ollama est installé sur votre PC, la version web dans le navigateur ne peut PAS l'utiliser pour des raisons de sécurité (blocage CORS).
Pour utiliser Ollama, vous DEVEZ télécharger et installer la version desktop de Cap'taine.

Cap'taine couvre les niveaux du CP à la 3ème (6 à 14 ans environ).

Comment ça marche :

  • Exercices pré-enregistrés : L'enfant choisit son niveau (CP, CE1, CE2... jusqu'à 3ème). Tous les niveaux sont accessibles librement.
  • Chatbot IA : Adapte la difficulté en temps réel (facile/moyen/difficile) selon les performances de l'enfant.

⚠️ Le système n'adapte PAS automatiquement le niveau scolaire (pas de passage auto CP→CE1). C'est vous ou l'enfant qui choisit le niveau dans les réglages.

Cap'taine couvre toutes les matières du primaire au collège :

  • Mathématiques : calcul, numération, géométrie, problèmes, mesures, algèbre, fractions
  • Français : grammaire, conjugaison, orthographe, vocabulaire, lecture
  • Sciences : SVT, physique-chimie, découverte du monde
  • Histoire-Géographie : programme complet primaire et collège
  • Langues étrangères : anglais, espagnol, allemand, italien (vocabulaire, grammaire de base)

Plus de 2000 exercices et 13+ mini-jeux éducatifs.

Oui. Les exercices et compétences sont alignés sur les programmes officiels français :

  • Cycle 2 : CP, CE1, CE2
  • Cycle 3 : CM1, CM2, 6ème
  • Cycle 4 : 5ème, 4ème, 3ème (collège)

Cependant, Cap'taine n'est pas un manuel scolaire : c'est un outil complémentaire pour s'entraîner et réviser.

Installation et configuration

  1. Téléchargez le fichier correspondant à votre système (Windows .exe, macOS .dmg, Linux .AppImage)
  2. Lancez l'installateur et suivez les instructions
  3. C'est prêt ! Aucune création de compte n'est nécessaire

Note Windows : Windows Defender peut afficher un avertissement car l'application n'est pas signée par Microsoft. Cliquez sur "Informations complémentaires" puis "Exécuter quand même".

Sans IA locale (Ollama) :

  • 4 Go de RAM
  • 500 Mo d'espace disque
  • Processeur dual-core récent

Avec IA locale (Ollama) :

  • 8 Go de RAM minimum (16 Go recommandé)
  • 10 Go d'espace disque (pour les modèles IA)
  • Processeur quad-core ou GPU compatible

Ollama est un logiciel gratuit qui permet de faire tourner des modèles d'IA (comme Llama 3, Mistral) directement sur votre ordinateur.

Avantages :

  • 100% local : aucune donnée ne quitte votre PC
  • Gratuit et sans limite
  • Pas besoin de compte ou de clé API

Inconvénients :

  • Nécessite un PC assez puissant (8+ Go RAM)
  • Téléchargement initial des modèles (~4 Go)

C'est optionnel : Cap'taine fonctionne très bien sans Ollama, avec la banque d'exercices intégrée.

Oui ! Si vous avez déjà une clé API pour OpenAI, Anthropic (Claude) ou Google (Gemini), vous pouvez la configurer dans Cap'taine.

Important : Dans ce cas, les conversations avec l'IA seront envoyées aux serveurs du fournisseur. Si la vie privée est votre priorité, utilisez Ollama.

Chaque fournisseur a ses propres tarifs (généralement quelques centimes par conversation).

Recommandations par âge :

  • Primaire (CP-CM2) : 15-20 min par jour, 30-45 min max par session
  • Collège (6ème-3ème) : 20-30 min par jour, jusqu'à 60 min par session

Cap'taine suggère automatiquement une pause après 25 minutes (technique Pomodoro) et détecte les signes de fatigue.

La régularité est plus importante que la durée : 15 min chaque jour vaut mieux que 2h le dimanche.

Vie privée et données

Sur votre ordinateur (stockage local) :

  • Profils utilisateurs (prénom, niveau)
  • Historique des exercices et réponses
  • Statistiques de progression
  • Préférences de l'application

Envoyé sur internet :

  • Avec Ollama (local) : RIEN
  • Avec OpenAI/Claude/Gemini : Les messages de conversation uniquement

Nous n'avons aucun serveur qui collecte vos données. Pas de télémétrie, pas d'analytics.

Toutes les données sont stockées localement sur votre ordinateur, dans le dossier de l'application :

  • Windows : %APPDATA%\captaine
  • macOS : ~/Library/Application Support/captaine
  • Linux : ~/.config/captaine

Ces données sont au format IndexedDB (base de données navigateur). Vous pouvez les supprimer à tout moment en désinstallant l'application.

Oui. La version Famille est 100% conforme car :

  • Aucune donnée personnelle n'est envoyée à nos serveurs (nous n'en avons pas)
  • Vous avez un contrôle total sur vos données
  • Vous pouvez supprimer toutes les données en désinstallant

Si vous utilisez une IA cloud (OpenAI, etc.), les données envoyées sont soumises à la politique de confidentialité du fournisseur.

Option 1 : Dans l'application, allez dans Paramètres → Données → Supprimer toutes les données

Option 2 : Désinstallez l'application (cela supprime tout)

Option 3 : Supprimez manuellement le dossier de données (voir question précédente)

Questions techniques

Non, et c'est volontaire. Cap'taine utilise des algorithmes heuristiques éprouvés :

  • SM-2 pour la répétition espacée (utilisé par Anki, SuperMemo)
  • Règles expertes pour la détection de patterns d'erreurs
  • Analyse de métriques pour l'adaptation de difficulté

Ces méthodes sont plus fiables qu'un ML mal entraîné et ne nécessitent pas des milliers d'exemples. Elles sont parfaitement adaptées à l'apprentissage scolaire.

L'IA (Ollama/OpenAI) est utilisée uniquement pour la génération de contenu et les explications, pas pour l'adaptation.

Plusieurs raisons :

  • Vie privée : Une app desktop stocke tout localement, sans serveur
  • Mode hors-ligne : Fonctionne sans connexion internet
  • Intégration Ollama : Peut communiquer avec l'IA locale
  • Performance : Accès direct au système de fichiers pour l'OCR

Nous travaillons à obtenir une signature de code pour les versions futures.

  • Vérifiez qu'aucun antivirus n'interfère
  • Essayez de supprimer le cache (Paramètres → Données → Vider le cache)
